    Sqlite client

    An sqlite client library with a pure, functional interface. The goal of the library is to provide a safe wrapper around the Java JDBC API, showcasing some of the basic functional features of Scala and how they can be used to make Java code more typesafe and pleasant to write.

    Installation

    Not yet released on Maven.

    With sbt:

    libraryDependencies += "TBA"

    With ammonite:

    import $ivy.`TBA`

    Imports

    import sqlite._

    Usage

    Refer to the guide.

    Compatibility

    Intended to be used with Scala 2.13 and above.
